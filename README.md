# Финансовый ИИ-навигатор

React-приложение в стиле тёмного AI-чата для расчёта накоплений на машину или любую другую цель.

## Что внутри

- React + Vite.
- Тёмная адаптивная тема.
- Чатовый ввод свободным текстом.
- Мок API в `src/services/api.js`.
- Комментарий для подключения реального API: `// ЗАМЕНИТЕ НА РЕАЛЬНЫЙ API: тут ваш URL и ключ`.
- Календарь накоплений по месяцам.
- Панель аналитики с SVG donut-диаграммой.
- Финансовая защита:
  - safety_net не меньше 10% дохода;
  - предупреждение, если свободный остаток меньше 5% дохода;
  - красное предупреждение, если свободный остаток отрицательный.

## Как запустить

```bash
npm install
npm run dev
```

После запуска Vite откроет приложение на локальном адресе, обычно:

```bash
http://localhost:5173
```

## Где подключать реальный API

Файл:

```bash
src/services/api.js
```

Сейчас используется мок:

```js
const API_MODE = 'mock';
```

Для реального API поменяйте на:

```js
const API_MODE = 'real';
```

И заполните:

```js
const API_URL = 'https://your-api-url.example.com/v1/chat/completions';
const API_KEY = 'YOUR_API_KEY_HERE';
```

Важно: для production лучше не хранить API-ключ во фронтенде. Правильнее сделать backend/proxy endpoint, а уже там хранить ключ в `.env`.
